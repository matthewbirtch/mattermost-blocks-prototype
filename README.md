# Mattermost Interactive Blocks — Prototype

A design prototype for exploring interactive block components within Mattermost posts. Built with React, TypeScript, and Vite.

## What this is

This repo is a component sandbox used to prototype the visual design and interaction patterns for integration-driven post attachments — things like incident alerts, approval requests, form inputs, and data displays that appear inside Mattermost messages from connected services.

It includes three views:

- **Components** — individual UI components in isolation (buttons, inputs, selects, key/value pairs, metrics, etc.)
- **Post Samples** — hand-crafted example posts showing realistic component arrangements for integrations like PagerDuty, Salesforce, Jira, and others
- **Autogenerated Posts** — a larger set of example posts across 20+ integrations, including interactive form examples

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
